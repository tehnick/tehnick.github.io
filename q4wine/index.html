Q4Wine — packages for Debian GNU/Linux and Ubuntu

Created: 2009-10-09

Updated: 2015-01-27

Description:

This is a free translation of my notes and it may differ from original.

Q4Wine is remarkable graphic utility for automation and simplification of work with WINE.

General q4wine features:
* Can export QT color theme into wine colors settings
* Can easy work with different wine versions at same time
* Easy creating, deleting and managing prefixes (WINEPREFIX)
* Easy controlling for wine process
* Easy installer wizard for wine applications.
* Autostart icons support
* Easy cd-image use
* You can extract icons from PE files (.exe .dll)
* Easy backup and restore for managed prefixes
* Winetricks support
* And more...

Last stable release version: 1.0-r3 (2013-05-03), see changelog for more details.
   
deb packages:
Packages can be downloaded here. They are suitable for Debian GNU/Linux version >= 6.0 (Squeeze) and for Ubuntu version >= 10.04 (Lucid Lynx). Sources are also available ibid.

To add my packages archive in Debian GNU/Linux Squeeze or Wheezy: 
echo "deb http://ppa.launchpad.net/tehnick/q4wine/ubuntu lucid main" | sudo tee /etc/apt/sources.list.d/q4wine.list
To add my packages archive in Debian GNU/Linux Jessie or Sid: 
echo "deb http://ppa.launchpad.net/tehnick/q4wine/ubuntu precise main" | sudo tee /etc/apt/sources.list.d/q4wine.list
Import the key: 
sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-keys 437276613F6729E2
Users of Ubuntu >= 10.04 (Lucid Lynx) can add my PPA using a standard utility add-apt-repository: 
sudo add-apt-repository ppa:tehnick/q4wine
Updating the list of available packages: 
sudo apt-get update
Installing q4wine: 
sudo apt-get install q4wine
Or you can install q4wine-unstable (daily builds): 
sudo apt-get install q4wine-unstable
q4wine-unstable package is updated automatically from git repository of program. Script for update is something like that.

Bug reports about deb packages you may send to me. Bug reports and feature request to the program you may leave here.

Now q4wine package is available in official Debian >= 6.0 (Squeeze) and Ubuntu >= 10.10 (Maverick Meerkat) repositories.

Thanks to Alex_P unofficial builds of q4wine packages are now available for Debian (squeeze, wheezy, sid) and Ubuntu (lucid, maverick, natty, oneiric).
