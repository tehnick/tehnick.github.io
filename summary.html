Псевдо-резюме
Пек Борис Эдуардович (1987)

Место проживания: Российская Федерация, г. Санкт-Петербург

Образование

В 2004 году по окончании 11 класса средней школы №16 имени В.Д. Ревякина в г. Севастополе выданы аттестат о среднем (полном) общем образовании и золотая медаль.

В 2008 году по окончании 4 курса Физического Факультета СПбГУ присвоена степень бакалавра Прикладных Математики и Физики (ПМФ) и выдан диплом без отличия.

В 2010 году по окончании 6 курса Физического Факультета СПбГУ присвоена степень магистра Прикладных Математики и Физики (ПМФ) и выдан диплом без отличия.

Работа

Сентябрь 2010 – март 2011 : стажировка в качестве инженера по испытаниям в НИИЛЭИ. Организация: ООО "КБ Коммутационной Аппаратуры" (подразделение промышленной группы "Таврида Электрик"). Задачи: экспериментальная работа, связанная с вакуумными выключателями.

Март 2011 – октябрь 2013 : инженер в КБ Численного Моделирования. Организация: ООО "КБ Коммутационной Аппаратуры" (подразделение промышленной группы "Таврида Электрик"). Задачи: компьютерное моделирование и экспериментальная работа, связанные с вакуумными выключателями.

Декабрь 2013 – апрель 2014 : фриланс + удаленная работа на одного заказчика. Задачи: разработка и доработка небольших кроссплатформенных (GNU/Linux, MS Windows) программ на C++ и Qt.

Май 2014 – настоящее время : инженер-программист в отделе разработки ПО БПЛА. Организация: ООО "Специальный Технологический Центр". Задачи: разработка ПО для тестирования отдельных узлов и БПЛА в целом (ТК, ПСИ и т.п.), разработка симулятора БПЛА и др..

Хобби

С 2007 года – разработка СПО.
С 2010 года – cопровождение пакетов в Debian и Ubuntu в статусе Sponsored Maintainer.
С 2012 года – официальный Debian Maintainer.
С 2013 года – официальный Debian Developer.

Навыки

В области разработки программного обеспечения:
  • Базовые знания классических алгоритмов и структур.
  • Хорошее знание C++, отличное знание Bash.
  • Есть опыт создания кроссплатформенного (GNU/Linux, MS Windows, Android) коммерческого ПО с использованием библиотек Qt.
  • Есть опыт написания с нуля, доработки и сопровождения кроссплатформенных (GNU/Linux, MS Windows, macOS, FreeBSD) приложений с графическим (QtWidgets) и консольным (QtCore) интерфейсами, а так же Linux-специфичных приложений на чистом STL. Данные программы относятся к свободному ПО, их код доступен для анализа.
  • Есть опыт совместной разработки СПО с использованием Subversion и Git в больших и малых командах разработчиков.
Знание английского: удовлетворительное. Хватает на чтение документации, для бытового общения и регулярной переписки (с использованием словаря).

Область интересов:
  • C++ (C++14 и новее предпочтителен) и фреймворк Qt (Qt 5.9 и новее предпочтителен).
  • Кроссплатформенное ПО с пользовательским интерфейсом (графическим или консольным).
  • Клиентское ПО для мгновенного обмена сообщениями, аудио- и видеосвязи.
  • ПО для обмена файлами и для хранения файлов с использованием различных p2p-протоколов.
  • ПО для численного моделирования.
  • Контроль качества.
Важное примечание: я не заинтересован в разработке низкоуровневого ПО (драйверов для оборудования, модулей ядра и т.п.).
