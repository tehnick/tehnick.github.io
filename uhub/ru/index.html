uhub — пакеты для Debian GNU/Linux и Ubuntu

Создано: 2011-09-27

Обновлено: 2015-08-27

Описание:

uhub является высокопроизводительным p2p хабом для сетей ADC. Низкое потребление ресурсов позволяет ему обеспечивать связь нескольких тысяч пользователей на полноценном сервере, или же работать в качестве небольшого приватного хаба на встраиваемых устройствах.

uhub использует протокол Advanced Direct Connect и совместим с DC++, LinuxDC++, EiskaltDC++ и другими ADC клиентами.

Основные характеристики:
* Высокая производительность и низкое потребление памяти.
* Поддержка IPv4 и IPv6.
* Поддержка SSL/TLS (опционально).
* Поддержка расширенного контроля доступа.
* Простая настройка.

Версия последнего стабильного релиза: 0.5.0 (2014-11-17), подробности в журнале изменений.

Активность в проекте: network graphic, commits in master branch
deb-пакеты:
Собранные пакеты можно скачать здесь. Они подходят для Debian GNU/Linux >= 7.0 (Wheezy) и для Ubuntu >= 12.04 (Precise Pangolin). Там же доступны исходники.

Подключение этого PPA в Debian GNU/Linux Wheezy: 
sudo add-apt-repository -y 'deb http://ppa.launchpad.net/tehnick/tehnick/ubuntu precise main'
Подключение этого PPA в Debian GNU/Linux Jessie: 
sudo add-apt-repository -y 'deb http://ppa.launchpad.net/tehnick/tehnick/ubuntu trusty main'
Подключение этого PPA в Debian GNU/Linux Stretch: 
sudo add-apt-repository -y 'deb http://ppa.launchpad.net/tehnick/tehnick/ubuntu vivid main'
Подключение этого PPA в Debian GNU/Linux Sid: 
sudo add-apt-repository -y 'deb http://ppa.launchpad.net/tehnick/tehnick/ubuntu vivid main'
Подключение этого PPA в Ubuntu >= 12.04 (Precise Pangolin): 
sudo add-apt-repository ppa:tehnick/tehnick
Обновление списка доступных пакетов: 
sudo apt-get update
Установка uhub: 
sudo apt-get install -V uhub
Или вы можете установить uhub-unstable (ежедневные сборки): 
sudo apt-get install -V uhub-unstable
Пакет uhub-unstable обновляется атоматически из git репозитория программы. Cкрипт для обновления выглядит примерно так.

Сообщения об ошибках в deb-пакетах пишите мне. Багрепорты и пожелания к программе оставляйте здесь.

Пакет uhub теперь доступен в официальных репозиториях Debian >= 7.0 (Wheezy) и Ubuntu >= 12.04 (Precise Pangolin).
