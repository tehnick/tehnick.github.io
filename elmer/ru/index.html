Elmer — пакеты для Debian GNU/Linux и Ubuntu

Создано: 2012-09-27

Обновлено: 2014-04-17

Описание:

Elmer это опенсорсная программа для моделирования физических процессов. Она содержит модели динамики жидкости, структурной механики, электромагнетизма, теплопереноса, акустики и др.. Они описаны уравнениями в частных производных, которые Elmer решает методом конечных элементов (МКЭ).
git-зеркало:

Существует неофициальное git-зеркало svn репозитория Elmer. Оно синхронизируется каждые два часа с помощью git-svn. Скрипт: update_git-svn_mirror.sh. Теги для старых коммитов были сделаны простым скриптом: update_tags_list.sh.

Только у меня есть доступ на запись в этот git репозиторий. И я не использую master ветку для своих коммитов. Временные ветки используются для подготовки патчей в апстрим. Например: mailmap.

Сервис OpenHub использует этот git репозиторий для обработки статистики проекта, потому что оригинальный svn репозиторий имеет некоторые проблемы.
deb-пакеты:

Собранные пакеты можно скачать здесь. Там же доступны исходники.

Пользователи Ubuntu >= 12.04 (Precise Pangolin) могут добавить мой PPA, используя стандартную утилиту add-apt-repository: 
sudo add-apt-repository ppa:tehnick/elmerfem
Обновление списка доступных пакетов: 
sudo apt-get update
Установка elmer (ежедневные сборки): 
sudo apt-get install elmer
Пакет elmerfem обновляется атоматически из svn репозитория программы. Cкрипт для обновления: automatic_update_elmerfem.

Сообщения об ошибках в deb-пакетах пишите мне. Багрепорты и пожелания к программе оставляйте здесь.

Старые версии пакета elmerfem доступны в официальных репозиториях Debian >= 6.0 (Squeeze) и Ubuntu >= 10.04 (Lucid Lynx).
